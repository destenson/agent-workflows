--- BUG LOG (bugs) ---

This project keeps a BUGS.md at the root — its log of open, unfixed bugs. Standing instruction for this session:

- If you find a bug you do not fix (deferred or out of scope), record it in BUGS.md with the bugs skill so it is not lost.
- If you fix a bug that is already listed there, remove its entry so the log reflects only what is still open.

This is a standing instruction, not a task — your work this session is whatever the user asks. Read BUGS.md on demand if you need the current list of open bugs; it is not loaded here.
