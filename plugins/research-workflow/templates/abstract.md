# Abstract — {project title}

The shortest durable statement of what this research is about. Loaded into context every session as the framing anchor. Keep it current; when the question or hypothesis shifts, this is the first thing to update.

- **Question**: the one thing this research is trying to find out, stated so an outcome could answer it.
- **Hypothesis (live)**: the current best answer being tested, phrased to be falsifiable. Update as it is supported, narrowed, or falsified.
- **Why it matters**: what a credible answer changes — the decision or capability downstream of it.
- **Scope / non-goals**: what is deliberately in and out, so the work does not sprawl.
- **Status**: one line on where the project is right now (framing · proposing · running · synthesizing · decided).
