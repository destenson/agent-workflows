--- SESSION PROBE (agentic-workflow) ---

The durable artifacts above (SPEC.md, ASSUMPTIONS.md, DECISIONS.md, LESSONS.md) are this project's memory. If none were shown, the project is not yet initialized — run /project-docs-init before implementation work.

Before touching any code, summarize:
1. Current project state.
2. The active constraints and invariants.
3. Open / unvalidated assumptions (and which are high-cost).
4. The scope of the current PRP or task.

Do not write or modify any code until this summary is done. A summary that misstates the project state, misses a key constraint, or shows confused reasoning is the signal to stop and restart the session — discarding a session here costs almost nothing, and a confused opening is an early sign of one that will go badly.
